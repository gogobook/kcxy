# kcxy

Rust restful webapp with rocket, diesel, postgresql, and react.js.

### Start

```bash
$ git clone https://github.com/mcux/kcxy.git
$ cd kcxy
$ cargo install diesel_cli --no-default-features --features postgres
$ diesel setup
$ create-react-app webfront
$ cd webfront
$ yarn build
$ cd ..
$ cargo run
```
