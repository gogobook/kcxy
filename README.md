# kcxy

Rust restful webapp with rocket, diesel, postgresql, and vue.js.

### Start

```bash
$ git clone https://github.com/mcux/kcxy.git
$ cd kcxy
$ cargo install diesel_cli --no-default-features --features postgres
$ diesel setup
$ cd webfront
$ npm install
$ npm run build
$ cd ..
$ cargo run
```
